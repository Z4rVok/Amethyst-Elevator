# Amethyst Elevator

A simple and efficient Minecraft datapack that allows players to create vertical elevators using **amethyst blocks** — no redstone required.

---

## ✨ Features

* Jump to go up
* Sneak to go down
* Automatically detects the next floor
* Works in multiplayer
* No setup needed

---

## 📦 Installation

1. Download the datapack `.zip`
2. Move it to your world folder:

   ```
   .minecraft/saves/<your-world>/datapacks/
   ```
3. Reload the world or run:

   ```
   /reload
   ```

---

## 🎮 How to Use

* Stand on an **amethyst block**
* Press:

  * **Jump** → go up
  * **Sneak** → go down

The datapack will automatically teleport you to the next available floor in the same vertical column.

---

## 📌 Supported Versions

* Minecraft **1.20.2 – 1.21.4+**
