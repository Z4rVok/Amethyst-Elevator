========================================
  Amethyst Elevator - Datapack
========================================

Stand on an Amethyst Block and:
  - Press JUMP  → go UP   to the next amethyst block
  - Press SNEAK → go DOWN to the next amethyst block

Floors are detected automatically by scanning up/down
for the nearest amethyst block in the same column.

----------------------------------------
  COMMANDS (run in chat or command block)
----------------------------------------

  /function amethyst_elevator:config_bossbar_on
      Show the floor indicator bossbar (default)

  /function amethyst_elevator:config_bossbar_off
      Hide the floor indicator bossbar

----------------------------------------
  NOTES
----------------------------------------

  - Supports up to 100 simultaneous players
  - Pack format: 48 (Minecraft 1.21+)
  - Namespace: amethyst_elevator
